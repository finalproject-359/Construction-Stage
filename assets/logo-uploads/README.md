# Logo Upload Folder

Ilagay dito ang bagong logo file na ipapalit sa kasalukuyang logo.

## File name format
Gamitin ang format na ito:

`app-logo-vYYYYMMDD.ext`

### Rules
- `app-logo` = fixed prefix
- `vYYYYMMDD` = petsa ng version (hal. `v20260522`)
- `.ext` = file extension (`png`, `svg`, o `webp`)

### Examples
- `app-logo-v20260522.png`
- `app-logo-v20260601.svg`
- `app-logo-v20260615.webp`

## Recommended specs
- Preferred: **SVG** (para malinaw sa lahat ng screen sizes)
- If raster image: at least **512x512 px**
- Keep transparent background kung kailangan
